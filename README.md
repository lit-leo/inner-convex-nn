# inner-convex-nn
Extreme sparsity by Input Convex Neural Networks.

## Experiments
* FC + MNIST
* U-NET + LungCT
* ResNET + Cifar10
* ConvAE + CelebA(reduced to 40k images)
* AlexNET + CATS/DOGS
